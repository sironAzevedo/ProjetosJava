Registrar Serviço de Proteção - Acesso a Pessoas

  • Como acessar?
  • Como preencher?
  • O que acontece depois?

Como acessar?

Para incluir um Serviço de Acesso a Pessoas em um Agenciamento, o usuário deverá clicar no botão: Serviço de Proteção do Agenciamento. O sistema abrirá a aba de Serviços de Proteção do Agencimanto.

Como preencher?

O usuário deverá clicar no botão Adicionar e depois na opção Serviço de Acesso a Pessoas.

O sistema abrirá a tela de cadastro do serviço com os seguintes campos:

  • Data da Solicitação, campo obrigatório. O sistema deve sugerir a data atual com a possibilidade de alteração pelo usuário.
  • Local(ais) do acesso, campo obrigatório.
  • Tipo de Solicitação de Trânsito, campo obrigatório.
  • Tipo de Acesso, campo obrigatório.
  • Categoria, campo obrigatório.
    • Caso a categoria seja Prestadores de Serviço:
      • É obrigatório selecionar a Empresa do prestador de serviço.
      • O usuário poderá selecionar pesquisando dentre as Empresas “ativas” já cadastradas em “Empresas/Pessoas – Serviço de Proteção”.
      • Após selecionar a Empresa, o sistema exibirá o CNPJ sem permitir edição, apenas para consulta.
      • Caso existam funcionários cadastrados para a Empresa selecionada, o sistema deverá carregar automaticamente na lista de Pessoas.
      • É obrigatório informar a Atividade de bordo.
  • Nacionalidade, campo obrigatório.
  • Lista de Pessoas, campo obrigatório.
    • Nome: É obrigatório selecionar uma Pessoa. O usuário poderá selecionar pesquisando dentre as Pessoas “ativas” já cadastradas em “Empresas/Pessoas – Serviço de Proteção”.
    • Tipo de Documento: Após selecionar a Pessoa, o sistema exibirá o Tipo de Documento (caso exista no cadastro) sem permitir edição, apenas para consulta).
    • Nº Documento: Após selecionar a Pessoa, o sistema exibirá o Nº Documento (caso exista no cadastro) sem permitir edição, apenas para consulta.
    • CPF: Após selecionar a Pessoa, o sistema exibirá o CPF (caso exista no cadastro) sem permitir edição, apenas para consulta.
    • Responsável: Obrigatório quando for um Visitante.
    • Nascimento: Após selecionar a Pessoa, o sistema exibirá o Nascimento (caso exista no cadastro) sem permitir edição, apenas para consulta.
    • Nacionalidade: Após selecionar a Pessoa, o sistema exibirá a Nacionalidade (caso exista no cadastro) sem permitir edição, apenas para consulta.
    • Volumes: Informar a quantidade de volumes (bagagens) que a Pessoa carrega.
    • Descrição da bagagem: Informa a descrição das bagagens que a Pessoa carrega.
    • Obs.: Caso seja informada a quantidade de Volumes, será obrigatório informar a Descrição da bagagem e vice-versa.
  • Observação, campo obrigatório apenas quando o usuário estiver alterando a solicitação.

O que acontece depois?

Ao clicar no botão Salvar, o serviço é salvo na Lista de Serviços de Proteção do Agenciamento.

O usuário poderá imprimir os Formulários da Polícia Federal e da Receita Federal.
