Registrar Serviço de Proteção - Suprimento de Retirada de Lixo/Resíduos

  • Como acessar?
  • Como preencher?
  • O que acontece depois?

Como acessar?

Para incluir um Serviço de Retirada de Lixo/Resíduos em um Agenciamento, o usuário deverá clicar no botão: Serviço de Proteção do Agenciamento. O sistema abrirá a aba de Serviços de Proteção do Agencimanto.

Como preencher?

O usuário deverá clicar no botão Adicionar e depois na opção Serviço de Retirada de Lixo/Resíduos.

O sistema abrirá a tela de cadastro do serviço com os seguintes campos:

  • Data da Solicitação, campo obrigatório. O sistema deve sugerir a data atual com a possibilidade de alteração pelo usuário.
  • Empresa Requerente, campo obrigatório.
  • Empresa Responsável Transbordo, campo obrigatório. Cadastro de Empresas e Serviços.
  • Embarcação Transbordo, campo obrigatório. Carregado após a escolha da Empresa Responsável Transbordo.
  • Transportadora Rodoviária, campo obrigatório. Cadastro de Empresas e Serviços.
  • Tipo de Resíduo, campo obrigatório.
  • Caracterização, campo obrigatório.
  • Estado Físico, campo obrigatório.
  • Classif. Código (ABNT NBR 10.004), campo obrigatório.
  • Código ONU, campo obrigatório.
  • Quantidade de Resíduos, campo obrigatório.
  • Data/Hora Início da Operação, campo obrigatório.
  • Data/Hora Término da Operação, campo obrigatório.
  • Local de Armazenagem Temporária dos Resíduos, o L.O Nº e o CADRI, campos obrigatórios para Tipo de Resíduo Classe I.
  • Destino Final do Resíduo, o L.O Nº e o CADRI, campos obrigatórios.
  • Valor do Serviço, campo obrigatório.
  • Observação, campo obrigatório apenas quando o usuário estiver alterando a solicitação.

O que acontece depois?

Ao clicar no botão Salvar, o serviço é salvo na Lista de Serviços de Proteção do Agenciamento.

O usuário poderá imprimir os formulários da Solicitação para os tipos de resíduos Classe I e II.

O usuário poderá gerar o relatório Excel com as Solicitações de Serviços de Retirada de Lixo/Resíduos.
