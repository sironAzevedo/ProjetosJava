Operações - Emitir CTAC

  • Como acessar?
  • Como preencher?
  • O que acontece depois?

Como acessar?

Para incluir uma Documentação na Operação, o usuário deverá acessar a aba Operações e clicar no botão: Documentação da Carga.

Como preencher?

O documento CTAC é emitido para as operações de carga cabotagem e está associado a um ou mais produtos.

A área de navegação deve estar informada na aba “Informações Gerais”.

O código do CTAC deve estar informado no cadastro do produto ou no cadastro da documentação.

É obrigatório informar a Data de Emissão.

É obrigatório selecionar o Endereço de Desembarque. As informações complementares (nome a ser impresso, endereço, inscrição estadual e CNPJ) do porto de Desembarque devem estar cadastradas.

É obrigatório informar o porto de embarque. As informações complementares (nome a ser impresso, endereço, inscrição estadual e CNPJ) do porto de embarque devem estar cadastradas.

É obrigatório selecionar o Endereço de Embarque.

Não é obrigatório informar uma Observação.

O que acontece depois?

Se, na Documentação cadastrada, foram informados o CE e o CTAC, o sistema deve gerar somente um formulário com a lista de Produtos selecionados. Se, na Documentação cadastrada, não foram informados o CE e o CTAC, o sistema vai gerar um formulário para cada Produto selecionado.

Emissão do documento CTAC e registro da emissão do documento para acompanhamento (aba Documentos).
