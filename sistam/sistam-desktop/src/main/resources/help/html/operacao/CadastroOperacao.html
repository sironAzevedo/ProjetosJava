Registrar Operações

  • Como acessar?
  • Como preencher?
  • O que acontece depois?

Como acessar?

Para incluir uma Operação, o usuário deverá acessar a aba Operações e clicar no botão: Adicionar.

Como preencher?

É obrigatório informar a operação realizada, o produto, a quantidade estimada e a unidade de medida.

Não é obrigatório informar a data início, a data fim e o anexo único.

O que acontece depois?

Ao clicar no botão Salvar, a Operação cadastrada para o agenciamento.

Para a operação de Carga Exportação será gerada uma pendência do tipo “Bill of Lading (BL)”, se o agenciamento for tipo contrato = “TCP”.
