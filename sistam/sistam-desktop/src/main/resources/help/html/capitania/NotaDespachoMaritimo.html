Capitania - Nota de Despacho Marítimo

  • Como acessar?
  • Como preencher?
  • O que acontece depois?

Como acessar?

Para gerar o formulário de Nota de Despacho Marítimo, o usuário deverá clicar no botão Comunicação Capitania e clicar na opção: Nota de Despacho Marítimo.

Como preencher?

O sistema deve abrir a tela de “Informações Complementares“ com os campos abaixo para informar antes da impressão:

  • Informações Complementares:
    • REPRESENTANTE LEGAL: O usuário deve selecionar o representante;
    • ORGÃO DE DESPACHO: O usuário deve selecionar o orgão do despacho;
    • ATRACAÇÃO: O usuário deve selecionar a atarcação;
    • ISENTO: O usuário deve escolher a opção SIM ou NÃO;
    • CONDUZ CARGA: O usuário deve escolher a opção SIM ou NÃO;
    • DATA DO DOCUMENTO: O usuário deve informar se deseja imprimir com a Data no formulário. Não é obrigatório;
    • Nº DE OFICIAIS: O usuário deve informar o número de oficiais;

O que acontece depois?

Ao clicar em "Gerar formulário", o SISTAM vai gerar o PDF de acordo com as opções selecionadas no filtro.
