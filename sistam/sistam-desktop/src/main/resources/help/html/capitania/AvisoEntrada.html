Capitania - Aviso de Entrada

  • Como acessar?
  • Como preencher?
  • O que acontece depois?

Como acessar?

Para gerar o formulário de Aviso de Entrada , o usuário deverá clicar no botão Comunicação Capitania e clicar na opção: Aviso de Entrada.

Como preencher?

O sistema deve abrir a tela de “Informações Complementares“ com os campos abaixo para informar antes da impressão:

  • Informações Complementares:
    • DEFICIÊNCIA A SEREM RETIFICADAS NESTE PORTO: O usuário deve escolher a opção SIM ou NÃO;
    • TRANSPORTA CARGA NO CONVÉS: O usuário deve escolher a opção SIM ou NÃO;
    • TRANSPORTA CARGA PERIGOSA: O usuário deve escolher a opção SIM ou NÃO;
    • EMBARCAÇÃO GRANELEIRO OU COMBINADO: O usuário deve escolher a opção SIM ou NÃO.
    • LOCALIZAÇÃO DO PORTO: O usuário deve informar a localização.
    • DATA DO DOCUMENTO: O usuário deve informar se deseja imprimir com a Data no formulário. Não é obrigatório;

O que acontece depois?

Ao clicar em "Gerar formulário", o SISTAM vai gerar o PDF de acordo com as opções selecionadas no filtro.
