Como acessar?

Para gerar o Relatório de Visitas, o usuário deverá acessar o menu Relatórios e clicar na opção: Relatório de Visitas.

Como preencher?

O sistema deve abrir uma tela com os seguintes filtros:

  • Agência: O usuário deve selecionar uma Agência dentre aquelas que o mesmo está associado. Campo obrigatório.
  • Porto: Após selecionar uma Agência, o sistema vai filtrar os portos. O usuário pode escolher filtrar por um Porto específico ou todos (deixar em branco). Campo não obrigatório.
  • Agente: Após selecionar uma Agência, o sistema vai filtrar os usuários que tem perfil de Agente Marítimo e que estão associados a essa agência. O usuário pode escolher filtrar por um Agente específico ou todos (deixar em branco). Campo não obrigatório.
  • Responsável pelo Custo: O usuário poderá selecionar um, ou mais de um ou por todos responsaveis pelo custo as opções: Todos,E&P, Petrobras, Transpetro,Gás e Energia e Sem Custo. Não marcando nenhuma das opções, o sistema deve buscar todos os tipos. Campo não obrigatório.
  • Tipo de Transporte: O usuário poderá selecionar as opções: Terrestre, Marítimo, Aquaviário, Aéreo e Fluvial para filtrar as visitas conforme o Tipo de Transporte. Não marcando nenhuma das opções, o sistema deve buscar todos os tipos. Campo não obrigatório.
  • Período: Datas no formato dia/mês/ano para filtrar as Visitas a partir da data de Início. Campo obrigatório.

O que acontece depois?

Ao clicar no botão “Gerar”, o SISTAM abrir o relatório de Visitas no Microsoft Excel.
