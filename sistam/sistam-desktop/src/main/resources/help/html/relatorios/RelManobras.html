Como acessar?

Para gerar o Relatório de Manobras, o usuário deverá acessar o menu Relatórios e clicar na opção: Relatório de Manobras.

Como preencher?

O sistema deve abrir uma tela com os seguintes filtros:

  • Agência Marítima: O usuário deve selecionar uma Agência dentre as que o mesmo está associado. Campo obrigatório.
  • Porto: Após selecionar uma Agência, o sistema vai filtrar os portos. O usuário pode escolher filtrar por um Porto específico ou todos. Campo não obrigatório.
  • Responsável pelo Custo: O usuário poderá selecionar um, ou mais de um ou por todos responsaveis pelo custo as opções: Todos,E&P, Petrobras, Transpetro,Gás e Energia e Sem Custo(nova opção da tela de manobra) para filtrar as manobras conforme o responsável pelo custo. Campo não obrigatório.
  • Período: Datas no formato dia/mês/ano para filtrar as Manobras Encerradas e as Canceladas Fora do Prazo pela Data de Início da Manobra. Campo obrigatório.
  • Tipo de Contrato: O usuário terá a opção de escolher um tipo de contrato ou todos.

O que acontece depois?

Ao clicar no botão “Gerar”, o SISTAM abrir o relatório de Manobras no Microsoft Excel.
