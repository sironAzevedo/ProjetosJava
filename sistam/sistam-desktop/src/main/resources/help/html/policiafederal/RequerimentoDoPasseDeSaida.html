Polícia Federal - Requerimento do Passe de Saída

  • Como acessar?
  • Como preencher?
  • O que acontece depois?

Como acessar?

Para gerar o formulário da Polícia Federal - Requerimento do Passe de Saída, o usuário deverá clicar no botão Polícia Federal e clicar na opção: Requerimento do Passe de Saída. Observação: Esse formulário está disponível somente para a Agência Salvador.

Como preencher?

O sistema deve abrir a tela de “Informações Complementares“ com os campos abaixo para informar antes da impressão:

  • Nº do Passe de Entrada: O usuário deve informar se deseja imprimir o Nº do Passe de Entrada no formulário. É obrigatório;
  • Operação: O usuário deve informar se deseja imprimir a Operação no formulário. É obrigatório;
  • Data da Entrada no Porto: O usuário deve informar se deseja imprimir a Data da Entrada no Porto no formulário. É obrigatório;
  • Tipo de Requerimento: O usuário deve informar uma das opções: Emissão, Reemissão ou Cancelamento para imprimir no formulário. É obrigatório;
  • Justificativa: O usuário deve informar se deseja imprimir uma Justificativa no formulário. Não é obrigatório;
  • Data do Documento: O usuário deve informar uma data se deseja imprimir a Data do Documento no formulário. Não é obrigatório;
  • Representante Legal: O usuário deve informar se deseja imprimir o Representante Legal no formulário. O sistema deve carregar os representantes legais da Agência Salvador que estão ATIVOS. É obrigatório;

O que acontece depois?

Gerar o formulário conforme campos preenchidos antes da impressão.

Após gerar o formulário, o sistema deve criar o documento “Requerimento do Passe de Saída da Polícia Federal” na aba Documentos do agenciamento.
