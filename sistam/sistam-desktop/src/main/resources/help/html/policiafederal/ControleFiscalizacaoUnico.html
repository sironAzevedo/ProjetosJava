Polícia Federal - Controle de Fiscalização Único

  • Como acessar?
  • Como preencher?
  • O que acontece depois?

Como acessar?

Para gerar o formulário da Polícia Federal - Controle de Fiscalização Único, o usuário deverá clicar no botão Polícia Federal e clicar na opção: Controle de Fiscalização Único. Observação: Esse formulário não está disponível para a Agência Salvador.

Como preencher?

O sistema deve abrir a tela de “Informações Complementares“ com os campos abaixo para informar antes da impressão:

  • Caso a agência marítima seja diferente de Manaus:
    • PASSE DE ENTRADA: O usuário deve selecionar se deseja imprimir o formulário com essa opção marcada;
    • PASSE DE SAÍDA: O usuário deve selecionar se deseja imprimir o formulário com essa opção marcada;
    • REGISTRO DE CABOTAGEM: O usuário deve selecionar se deseja imprimir o formulário com essa opção marcada. O usuário deve escolher se é um Registro de Cabotagem de Entrada ou Saída.
  • Caso a agência marítima seja Manaus:
    • SOLICITAÇÃO PASSE DE ENTRADA: O usuário deve selecionar se deseja imprimir o formulário com essa opção marcada;
    • SOLICITAÇÃO PASSE DE SAÍDA: O usuário deve selecionar se deseja imprimir o formulário com essa opção marcada;
    • COMUNICAÇÃO DE CAB./LONG. C: O usuário deve selecionar se deseja imprimir o formulário com essa opção marcada. O usuário deve escolher se é uma Comunicação de Entrada ou Saída;
    • PEDIDO DE VISITA: O usuário deve selecionar se deseja imprimir o formulário com essa opção marcada.
  • CNPJ Armador: O usuário deve informar se deseja imprimir o CNPJ do Armador no formulário. Não é obrigatório;
  • Agente Armador: O usuário deve informar se deseja imprimir o Agente Armador no formulário. Não é obrigatório;
  • Escalas: O usuário deve informar se deseja imprimir as Escalas no formulário. Não é obrigatório;
  • Produto da Carga: O usuário deve informar se deseja imprimir o Produto da Carga no formulário. Não é obrigatório;
  • Trânsito: O usuário deve informar se deseja imprimir a quantidade de pessoas em Trânsito no formulário. Não é obrigatório;
  • Clandestinos/Impedidos: O usuário deve informar se deseja imprimir a quantidade de pessoas Clandestinas/Impedidas no formulário. Não é obrigatório;
  • Porto de Entrada no Brasil: O usuário deve informar se deseja imprimir o Porto de Entrada no Brasil no formulário. Não é obrigatório;
  • Data: O usuário deve informar se deseja imprimir com a Data no formulário. Não é obrigatório;
  • Representante Legal: Esse campo mostra os Representantes Legais da Agência; Obs. O cliente solicitou esse campo, porém não é exibido no formulário.

O que acontece depois?

Gerar o formulário conforme campos preenchidos antes da impressão.

Caso exista a Taxa “FUNAPOL / Polícia Federal (Entrada)”, o formulário deve “marcar” a opção “Taxa Paga” no Passe de Entrada;

Caso exista a Taxa “FUNAPOL / Polícia Federal (Saída)”, o formulário deve “marcar” a opção “Taxa Paga” no “Passe de Entrada” e na “Solicitação Passe de Saída”.

  • Após gerar o formulário, o sistema deve criar os seguintes Documentos na aba Documentos do agenciamento:
    • Passe de Entrada ou Solicitação Passe de Entrada - O sistema deve gerar um novo Documento do tipo “Passe de Entrada da Polícia Federal”;
    • Passe de Saída ou Solicitação Passe de Saída - O sistema deve gerar um novo Documento do tipo “Passe de Saída da Polícia Federal”;
    • Registro de Cabotagem ou Comunicação de Cab./Longo C de Entrada - O sistema deve gerar um novo Documento do tipo “Controle de Fiscalização Único (entrada)”;
    • Registro de Cabotagem ou Comunicação de Cab./Longo C de Saída - O sistema deve gerar um novo Documento do tipo “Controle de Fiscalização Único (saída)”;
    • Caso a Data Oficial de Chegada esteja preenchida no agenciamento, o sistema deve gerar uma pendência no agenciamento do tipo “Controle de Fiscalização Único de Entrada”;
    • Caso a Data Estimada de Saída esteja preenchida no agenciamento, o sistema deve gerar uma pendência no agenciamento do tipo “Controle de Fiscalização Único de Saída”.
