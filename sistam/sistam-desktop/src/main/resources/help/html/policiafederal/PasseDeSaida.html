Polícia Federal - Passe de Saída

  • Como acessar?
  • Como preencher?
  • O que acontece depois?

Como acessar?

Para gerar o formulário da Polícia Federal - Passe de Saída, o usuário deverá clicar no botão Polícia Federal e clicar na opção: Passe de Saída. Observação: Esse formulário está disponível somente para a Agência Salvador.

Como preencher?

O sistema deve abrir a tela de “Informações Complementares“ com os campos abaixo para informar antes da impressão:

  • Escalas: O usuário deve informar se deseja imprimir as Escalas no formulário. Não é obrigatório;
  • Carga: O usuário deve informar se deseja imprimir a Carga no formulário. Não é obrigatório;
  • Em Trânsito:
    • Clandestinos/Impedidos: O usuário deve informar a quantidade de Clandestinos/Impedidos em Trânsito se deseja imprimir no formulário. Não é obrigatório;
    • Outros: O usuário deve informar a quantidade de Outros em Trânsito se deseja imprimir no formulário. Não é obrigatório;
  • Desembarque neste Porto:
    • Tripulantes: O usuário deve informar a quantidade de Tripulantes que desembarcaram no Porto, se deseja imprimir no formulário. Não é obrigatório;
    • Passageiros: O usuário deve informar a quantidade de Passageiros que desembarcaram no Porto se deseja imprimir no formulário. Não é obrigatório;
    • Clandestinos/Impedidos: O usuário deve informar a quantidade de Clandestinos/Impedidos que desembarcaram no Porto se deseja imprimir no formulário. Não é obrigatório;
    • Outros: O usuário deve informar a quantidade de Outros que desembarcaram no Porto se deseja imprimir no formulário. Não é obrigatório;
  • Embarque neste Porto:
    • Tripulantes: O usuário deve informar a quantidade de Tripulantes que embarcaram no Porto, se deseja imprimir no formulário. Não é obrigatório;
    • Passageiros: O usuário deve informar a quantidade de Passageiros que embarcaram no Porto se deseja imprimir no formulário. Não é obrigatório;
    • Clandestinos/Impedidos: O usuário deve informar a quantidade de Clandestinos/Impedidos que embarcaram no Porto se deseja imprimir no formulário. Não é obrigatório;
    • Técnicos: O usuário deve informar a quantidade de Técnicos que embarcaram no Porto se deseja imprimir no formulário. Não é obrigatório;
  • GRU: O usuário deve informar uma das opções: Recolhida, Isento, Recolhida em Juízo ou Outros para imprimir no formulário. Não é obrigatório;
  • Data do Documento: O usuário deve informar uma data se deseja imprimir a Data do Documento no formulário. Não é obrigatório;

O que acontece depois?

Gerar o formulário conforme campos preenchidos antes da impressão.

  • A coluna Total deverá conter:
  • Tripulantes: Informação “Tripulantes” na Saída da Embarcação do Agenciamento;
  • Passageiros: Informação “Passageiros” na Saída da Embarcação do Agenciamento;
  • Impedidos/Clandestinos: ( Qtde informada em “Em Trânsito” - Qtde informada em “Chegada da Embarcação”) + Qtde informada em “Entrada da Embarcação”;
  • Técnicos: ( Qtde informada em “Em Trânsito” - Qtde informada em “Chegada da Embarcação”) + Qtde informada em “Entrada da Embarcação”.

Após gerar o formulário, o sistema deve criar o documento “Passe de Saída da Polícia Federal” na aba Documentos do agenciamento.

Caso a Data Estimada de Saída esteja preenchida no agenciamento, o sistema deve gerar a pendência no agenciamento: “Registro de Cabotagem ou Passe de Saída PF”;
